About

LogVision

LogVision is a streaming log analysis & intrusion detection system. It's a simple and comprehensive example of a streaming data analysis & intrusion detection workflow. It's designed and developed by one developer, and it took him 100 days(nights) to build from scratch. For more information, you can refer to the project GitHub page.

The current prototype is not built for production use.

LogVision is built upon:

Apache Flume, Apache Kafka, Apache Spark, MongoDB, Redis, Flask, Socket.IO, Echarts
